# DocPilot

> Enterprise AI Document Assistant powered by Retrieval-Augmented Generation (RAG).

---

## Features

- 📄 Upload PDF documents
- 📝 Upload DOCX documents
- 📃 Upload TXT documents
- 🔎 Semantic Search using FAISS
- 🤖 AI Question Answering
- 📚 Multi-document Retrieval
- 📌 Source Citations
- ⚡ FastAPI Backend
- ⚛ React Frontend

---

## Tech Stack

### Backend

- FastAPI
- LangChain
- FAISS
- HuggingFace
- Python

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- ShadCN UI
